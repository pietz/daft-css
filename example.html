How It Works

The simplified color system uses just one CSS variable to control the entire theme:

:root {
    --daft-theme: #2060df;  /* Any CSS color! */
}

/* Also works with: */
--daft-theme: rgb(32, 96, 223);
--daft-theme: hsl(220, 85%, 55%);
--daft-theme: tomato;

All other colors are automatically calculated from these base values:

  • Hover states are dynamically darkened
  • Focus states use transparency
  • Borders and backgrounds use color-mix()
  • Text colors adapt to maintain contrast

Benefits

  • Single CSS file instead of 19 theme files
  • Infinite color possibilities instead of predefined themes
  • ~90% smaller CSS footprint
  • Easy customization with just 1 variable
